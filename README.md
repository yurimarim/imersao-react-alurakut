## Clonar o projeto

```bash
# Clone o rep
$ git clone https://github.com/yurimarim/imersao-react-alurakut.git

# Entre na pasta raiz
$ cd SoulsKut

# Instale as depêndencias
$ yarn

# Rode o projeto(acesse http://localhost:3000)
$ yarn start
```
